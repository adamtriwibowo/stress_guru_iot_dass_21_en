IoT Information System for Autism Teacher Stress Assessment using the DASS-21 Method

About This System

A web-based platform that uses Internet of Things (IoT) technology and the DASS-21 psychological instrument to measure stress levels in autism companion teachers. This system provides a comprehensive solution for monitoring and analyzing the psychological condition of teachers in an easy, standardized, and accurate way — combining physiological sensor data with self-report questionnaire data.

How the System Works
1
Respondent Data Input

The teacher fills in personal data: name, age, gender, and teaching experience

Example: R55 — Siti Aminah, 35 years old, Female, 10 years of teaching experience
2
IoT Sensor Measurement

Sensors measure the teacher's heart rate and body temperature over 2 tests

Example Sensor Data:
• Heart Rate: Test 1 = 120 BPM, Test 2 = 118 BPM
• Body Temperature: Test 1 = 31.5°C, Test 2 = 31.0°C
3
DASS-21 Questionnaire

The teacher answers 7 questions about their psychological condition over the past week

Example Question: "I tend to feel easily agitated"
Options: 0 = Never  ·  1 = Sometimes  ·  2 = Often  ·  3 = Almost Always
Example Answer: 2 (Often)
4
Automatic Score Calculation

The system calculates the DASS-21 score according to international psychological standards

Calculation:
• Raw Score = 2+1+3+2+2+1+2 = 13
• Final Score = 13 × 2 = 26
• Category = 3 — Severe
5
Results & Analytics Dashboard

Test results are displayed with visualizations and an interactive analytics dashboard

Dashboard displays:
• Complete statistics (total respondents, average score)
• Stress distribution charts (bar chart, pie chart, line chart)
• Full test results table with stress levels

IoT Sensor Integration (ESP32/Arduino)

Example code to send sensor data from an ESP32 device to the server:

C++ / Arduino 
#include <WiFi.h>
#include <HTTPClient.h>

void sendSensorData() {
  int   heartRate   = 118;      // Read from MAX30100 sensor
  float temperature = 31.5;   // Read from DS18B20 sensor

  HTTPClient http;
  http.begin("http://192.168.1.100:5000/api/iot/sensor");
  http.addHeader("Content-Type", "application/json");

  String data = "{\"heart_rate\":" + String(heartRate) +
                ",\"temperature\":" + String(temperature) + "}";

  int response = http.POST(data);
  http.end();

  Serial.println("Data sent: " + data);
}
